## üìä Using the Application
1. **Connect Your ESP32**: Make sure your ESP32 device is powered on and connected to the same Wi-Fi network as your computer.

2. **Configure Settings**: Open the application and access the settings menu. Here, you can set thresholds for air quality alerts and select the sensors you want to use.

3. **Start Monitoring**: Once configured, click on the "Start Monitoring" button. You will see real-time data displayed in the dashboard.

4. **Access Data via API**: If you're interested in more technical aspects, use the provided REST API to access your air quality data programmatically. Detailed API documentation can be found in the repository.

5. **Review Your Data**: Use the dashboard to view historical data, trends, and visual representations of air quality over time.
